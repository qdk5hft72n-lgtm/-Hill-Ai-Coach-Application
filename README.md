## What This Is

A vibe-coded application page demonstrating how I actually work with AI — not just describe it. Built using Claude as the execution engine, structured as a multi-file static site to show composition and code organization.

## Structure

```
index.html          → Main page (4 sections: Cover, Resume, Builds, Learning)
css/styles.css      → Full design system — layout, nav, hero, components, animations
js/builds.js        → AI builds data + card renderer
js/main.js          → Scroll reveals, nav behavior, parallax, micro-interactions
README.md           → This file
```

## Sections

1. **Cover Letter** — Answering the prompt: "Describe an AI workflow you built that solved a real problem"
2. **Resume** — Certifications, experience, AI stack, technical skills, active builds
3. **AI Builds** — 8 real workflows and apps in active use
4. **Learning Velocity** — Last AI feature absorbed: NotebookLM Podcast Mode

## Running Locally

No build step. Open `index.html` in any browser. Requires internet for Google Fonts.

## Hosting

Deploy to any static host:
- **Netlify:** Drag folder into netlify.com/drop
- **GitHub Pages:** Push to repo → Settings → Pages → Deploy from `main`
- **Vercel:** `vercel deploy` from this directory
